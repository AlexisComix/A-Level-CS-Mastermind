Mastermind

What is Mastermind?

Mastermind is a board game that, usually, uses coloured pegs and is played with two players. Comparisons can be drawn to the more modern Wordle, however the game Mastermind has more mathematical complexity behind it as it does not explicitly show which peg or counter (or in the case of Wordle, letter) is correct/in the wrong place.

Main Menu and Title Screen

This is the Main Menu and Title Screen. It is navigated by the arrow keys, with option select bound to Z and going back a menu bound to Esc. The menu is designed to be intuitive and user-friendly. It only shows a few options at once for the user, and the indication of the selection is small but clear - deliberately unobtrusive. It contains an option to play the game, an option to see the controls, and an option to exit the game using controls inside the game rather than pressing the 'x'. The whole application is usable with just the keyboard, which works well for this retro-style look.
